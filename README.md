# Zerodha Project

This repository contains two React projects used for a Zerodha-like dashboard and landing frontend:

- `dashboard/` - Vite-based React dashboard (use `npm run dev` or `npm start`)
- `frontend/` - Create React App-based landing pages (use `npm start`)

Quick start:

1. Install dependencies for each project:

```bash
cd dashboard && npm install
cd ../frontend && npm install
```

2. Run the dashboard locally:

```bash
cd dashboard
npm run dev
```

3. Run the landing frontend locally:

```bash
cd frontend
npm start
```
